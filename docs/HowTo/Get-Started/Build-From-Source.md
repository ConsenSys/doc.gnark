## Installation on Linux / Unix / macOS

### Clone the gnark Repository

Clone the `Consensys/gnark` repository:

```bash
git clone https://github.com/Consensys/gnark.git
```

### Build gnark

After cloning, go to the `gnark` directory.

```bash
cd gnark
```

Build gnark
